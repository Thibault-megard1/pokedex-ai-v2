# Battle Backgrounds

This folder contains battle background images used in the battle scene.

## Required Files

Place the following image files in this directory:

- **grass.png** - Grassy field background (for grass encounters)
- **cave.png** - Dark cave background (for cave encounters)  
- **route.png** - Open field/route background (for route encounters)

## Specifications

- **Format**: PNG with transparency support
- **Recommended size**: 800×600 pixels (will be scaled to fit screen)
- **Style**: Pokémon GBA/DS style backgrounds

## Sources

You can find battle backgrounds from:
- Pokémon GBA ROM sprites (The Spriters Resource)
- Free Pokémon fan game assets
- Create custom backgrounds matching the Pokémon aesthetic

## Fallback

If a background image is missing, the game will use a simple gradient fallback (sky blue to grass green).
