# Pokédex AI (Next.js + DB locale)

## Prérequis
- Node.js (LTS)

## Installation
```bash
npm install
npm run dev
```

Ouvre http://localhost:3000

## Stockage local
Tout est stocké dans `./data/` :
- users.json (comptes)
- sessions.json (sessions)
- teams.json (équipes)
- pokemon-cache/ (cache PokéAPI)
